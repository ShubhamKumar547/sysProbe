# sysProbe 

**sysProbe** is a lightweight, Bash-based system resource monitoring tool designed for Linux distributions (optimized for Ubuntu).


**Dependencies**
1. curl
2. uptime
3. free 
4. df 
5. lscpu 
6. grep 
7. awk 
8. sed



##  Prerequisites

* **OS:** Linux 
* **Shell:** Bash 
* **Permissions:** Terminal access with execution rights.

##  Installation

You can set up `sysProbe` in seconds.

1.  **Clone the repository:**
    ```bash
    https://github.com/ShubhamKumar547/sysProbe.git
    ```

2.  **Navigate to the directory:**
    ```bash
    cd sysProbe
    ```

3.  **Make the script executable:**
    ```bash
    chmod +x monitor.sh
    ```

##  Usage

To start monitoring your system, simply run the script from your terminal:

```bash
./monitor.sh
